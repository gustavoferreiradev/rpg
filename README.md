## 💻 Sobre o projeto

**Simulador de RPG** é um protótipo de um jogo de RPG onde foram programados os movimentos de um personagem em 2 dimensões.

---

## ⚙️ Funcionalidades

- [x] Simular movimentos 2D com as teclas de seta e W, A, S, e D do teclado.
- [x] Validação para evitar que o personagem saia do mapa.

---

## 🚀 Como executar o projeto


### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com), [NodeJS](https://nodejs.org), [Yarn Package Manager](https://yarnpkg.com/). 
Além disto é bom ter um editor para trabalhar com o código como [VSCode](https://code.visualstudio.com/)

#### 🧭 Rodando a aplicação web

```bash

# Clone este repositório
$ git clone git@github.com:gustavoferreiradev/rpg.git

# Acesse a pasta do projeto no seu terminal/cmd
$ cd rpg

# Instale as dependências com o comando
$ yarn install

# Rode o projeto com o comando
$ yarn start

# A aplicação será aberta na porta:3000 - acesse http://localhost:3000

```
